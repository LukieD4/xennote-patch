## English
XenNote is a Fabric mod which adds xenharmonic support to Minecraft.

Supporting game versions: 1.21~1.21.3

Require: Cloth Config API

![](en.png)

For details, see the [Wiki](https://github.com/zhenlige/xennote/wiki).
